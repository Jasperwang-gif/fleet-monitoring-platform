# Fleet Monitoring Platform

## 📌 Project Overview

A full-stack fleet monitoring system to track vehicle telemetry, generate alerts, and visualize fleet status.

## 🛠 Tech Stack

* Backend: Java Spring Boot
* Frontend: React
* Database: PostgreSQL

## 🚧 Planned Features

* Vehicle management
* Telemetry ingestion
* Alert system (overspeed, low battery)
* Dashboard with fleet statistics
* Authentication (JWT)
